Política de Privacidade

Esta Política de Privacidade descreve como o CheckInfra coleta, utiliza e protege informações durante o uso do sistema. O CheckInfra é um projeto acadêmico e tecnológico voltado ao diagnóstico sanitário de banheiros escolares, com foco em apoio à gestão pública.

1. Quais dados são coletados

O CheckInfra pode coletar os seguintes dados:

  • Nome informado do avaliador;
  • Nome da escola avaliada;
  • Respostas do checklist técnico;
  • Registro fotográfico enviado pelo usuário;
  • Data e horário da avaliação;
  • Dados técnicos de navegação de forma anonimizada (Google Analytics).

O sistema não coleta dados sensíveis, como CPF, endereço residencial, dados biométricos ou informações pessoais de alunos.

2. Finalidade do uso dos dados

Os dados coletados são utilizados exclusivamente para:

  • Gerar diagnósticos técnicos sobre condições sanitárias;
  • Apoiar a priorização de intervenções em infraestrutura escolar;
  • Produzir relatórios técnicos e indicadores;
  • Melhorar a experiência e o funcionamento do sistema.

3. Base legal (LGPD)

O tratamento dos dados é realizado com base no legítimo interesse e na finalidade pública, conforme a Lei Geral de Proteção de Dados (Lei nº 13.709/2018), especialmente para fins de diagnóstico, planejamento e gestão da infraestrutura educacional.

4. Armazenamento e segurança

Os dados podem ser temporariamente armazenados no próprio dispositivo do usuário quando o sistema é utilizado em modo offline. A sincronização ocorre apenas quando há conexão com a internet.

O CheckInfra adota boas práticas técnicas para reduzir riscos de acesso não autorizado, perda ou uso indevido das informações.

5. Compartilhamento de dados

O CheckInfra não comercializa dados pessoais. As informações podem ser acessadas apenas por gestores ou equipes técnicas autorizadas, no contexto institucional do uso do sistema.

6. Direitos do usuário

O usuário pode, a qualquer momento:

  • Solicitar informações sobre os dados registrados;
  • Solicitar correção de informações;
  • Solicitar a exclusão de dados, quando aplicável.

7. Atualizações desta política

Esta Política de Privacidade pode ser atualizada para refletir melhorias no sistema ou adequações legais. Recomenda-se a consulta periódica desta página.
